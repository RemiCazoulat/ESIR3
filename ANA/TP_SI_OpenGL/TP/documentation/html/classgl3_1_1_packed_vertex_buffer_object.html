Detailed Description

template<typename FirstType, typename... OtherTypes>
class gl3::PackedVertexBufferObject< FirstType, OtherTypes >

A class simplifying the use of vertex buffers containing more than one information per vertex (for instance position, normal, texture coordinates...). The provided template types are the type of each informations associated with a vertex. The information is packed in a tuple in which each per-vertex information has an index. This index is the index in the list of provided template parameters.
