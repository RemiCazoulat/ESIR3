class  EditSceneGraph
 A Gui based on ImGui that displays a scene graph and offers the possibility to modify some states. This Gui is implemented as a visitor that have to be applied on the scene graph. This visitor keeps a state, thus it should be used on only one scene graph and remain alive between the rendered frames. This editor can be displayed in any window, it is your responsibility to create this window. More...
