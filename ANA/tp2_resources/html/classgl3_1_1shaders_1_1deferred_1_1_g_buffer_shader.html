Detailed Description

A shader populating a frame buffer that have the characteristics of a G-Buffer.

  • Color attachment 0 : positions (usually in world coordinates) [RGBA32F] (x,y,z,linear depth)
  • Color attachment 1 : normals (usually in world coordinates) [RGB32F]
  • Color attachment 2 : diffuse color [RGB16F]
  • Color attachment 3 : specular color [RGBA16F] (r,g,b,shininess)
  • Plus a depth and stencil buffer.

The frame buffer can be created by using the method createFrameBuffer

The inputs (see convention.txt)

  • in_position
  • in_normal
  • in_textureCoordinates

The uniforms (see convention.txt)

  • uni_mat_model
  • uni_mat_view
  • uni_mat_projection
  • uni_mat_normal
  • uni_diffuseTexture
  • uni_specularTexture
  • uni_material
