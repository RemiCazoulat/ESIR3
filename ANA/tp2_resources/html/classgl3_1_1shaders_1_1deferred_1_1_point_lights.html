Detailed Description

A shader program handling the rendering of point lights in deferred shading mode. Please note that the rendering is associated with light volumes (geometries). You have to render the volumes to light parts of the model. The shader program only use geometry instancing and the index of the instance corresponds to the index of the light in the light arrays.

Please, also note that it is your responsibility to link the uniforms with the g-buffer textures. More over, for this shader to work correctly, the following requirement must be satisfied:

  • The ouput frame buffer should have the g-buffer depth buffer.
  • The ouput frame buffer sould have a color buffer linked to attachment 0
  • The depth write should be disabled but the depth test should remain enabled (in order to cull light volumes if needed).
  • The blending mode should be additive because the lights should accumulate in the color buffer.

The inputs (see convention.txt) :

  • in_position
  • in_mat_model

The uniforms (see convention.txt) :

  • uni_positionTexture
  • uni_diffuseTexture
  • uni_specularTexture
  • uni_normalTexture
  • uni_viewport
  • uni_viewerPosition
  • uni_pointLights
