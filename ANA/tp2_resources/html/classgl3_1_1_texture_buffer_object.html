Detailed Description

template<class Type>
class gl3::TextureBufferObject< Type >

A class interfacing with a texture buffer object. This object can be used to provide large one dimensional arrays to a fragment shader. The provided generic type should be float, glm::vec2, glm::vec3 or glm::vec4.
