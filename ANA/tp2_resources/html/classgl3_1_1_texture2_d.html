gl3::Texture2D::Texture2D ( std::filesystem::path  textureFile,
size_t  channels = 4,
bool  generateMipmaps = false 
)
inline

Generates a new texture from an image file. The clamping is set to repeat by default. The interpolation is set to linear, except if generateMipmaps is true. In that case, the nim filter if set to linearMipmapLinear (it should be the best quality).

Parameters
textureFileThe texture file.
channelsThe number of channels in the texture (1 = red, 2 = red/green, 3 = red/green/blue, 4 = red/green/blue/alpha).
generateMipmapsif set to true generate mipmaps.
