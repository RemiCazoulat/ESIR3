Detailed Description

Computes the collision between two convex shapes. Shapes can be provided as meshes or as a set of points. In both cases, the procedure will test the collision between the convex hulls of the meshes or the set of points. The implemented algorithm is the Gilbert–Johnson–Keerthi distance algorithm.
